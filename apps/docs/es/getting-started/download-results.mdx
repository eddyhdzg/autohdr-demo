# Descarga Tus Resultados

Después de que AutoHDR procese tus fotos, puedes revisarlas lado a lado
y descargar las que te gusten.

## Tiempos de entrega

| Plan       | Entrega típica       |
| ---------- | -------------------- |
| Gratis     | Hasta 24 horas       |
| Pro        | Minutos              |
| Enterprise | Procesamiento prioritario |

## Revisa tus ediciones

1. Abre tu proyecto en el Studio
2. Usa la **comparación lado a lado** para ver el antes y después
3. Selecciona las fotos que deseas descargar
4. Haz clic en **Descargar**

<Info>
  Solo se te cobra por las fotos que descargas. Sube y previsualiza tantas
  como quieras sin costo.
</Info>

## Descarga masiva

Para sesiones grandes, usa la opción **Seleccionar Todo** para descargar tu
proyecto completo de una vez.
