# Crea Tu Cuenta

AutoHDR ofrece un plan gratuito con 10 descargas de fotos por mes. No se requiere tarjeta de crédito.

## Regístrate

1. Visita [AutoHDR](https://autohdr.vercel.app)
2. Haz clic en **Comenzar Gratis**
3. Inicia sesión con tu cuenta de Google
4. Estás listo para subir tus primeras fotos

<Info>
  Tu plan gratuito incluye procesamiento HDR estándar con entrega en 24 horas.
  Actualiza a Pro para procesamiento prioritario (la mayoría de las fotos en minutos).
</Info>

## Qué incluye el plan gratuito

| Función             | Plan Gratuito |
| ------------------- | ------------- |
| Fotos gratis al mes | 10            |
| Procesamiento HDR   | Estándar      |
| Tiempo de entrega   | 24 horas      |
| Tarifa por uso      | $0.80/foto    |
| Límite de subida    | Ilimitado     |
