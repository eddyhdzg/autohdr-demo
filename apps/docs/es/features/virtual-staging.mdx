# Staging Virtual

Transforma habitaciones vacías en espacios bellamente decorados. El staging
virtual de AutoHDR agrega muebles, decoración y accesorios realistas para ayudar
a los compradores a visualizar el espacio.

## Cuándo usar staging virtual

- Propiedades vacantes
- Construcción nueva sin muebles modelo
- Habitaciones que necesitan un cambio de estilo para las fotos del listado

## Tipos de habitaciones soportados

- Salas de estar
- Dormitorios
- Comedores
- Oficinas en casa
- Cocinas (staging de encimera)

## Cómo funciona

1. Sube tu foto de una habitación vacía
2. Selecciona **Staging Virtual**
3. AutoHDR agrega muebles y decoración que combinan con el estilo,
   tamaño e iluminación de la habitación
4. Revisa y descarga tu foto con staging

<Info>
  El staging virtual debe ser divulgado en tu listado según las directrices de NAR.
  Incluye una nota de que las fotos son "virtualmente amuebladas" en la descripción de tu listado.
</Info>
