# Eliminación de Objetos

Limpia tus fotos de bienes raíces eliminando elementos no deseados como
botes de basura, autos, letreros de jardín o pertenencias personales.

## Eliminaciones comunes

- Botes de basura y contenedores de reciclaje
- Autos en las entradas
- Letreros de bienes raíces de agentes competidores
- Fotos personales y desorden
- Equipo de construcción
- Mangueras y herramientas de jardín

## Cómo solicitar eliminación de objetos

Al subir tus fotos, marca los elementos que deseas eliminar. La IA de AutoHDR
rellena el fondo de forma natural usando el contexto circundante.

<Warning>
  La eliminación de objetos funciona mejor para objetos pequeños con un fondo claro.
  Eliminar objetos grandes (como un auto que ocupa el 50% del encuadre) puede
  producir resultados menos naturales.
</Warning>
