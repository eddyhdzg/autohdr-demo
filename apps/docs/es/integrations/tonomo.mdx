# Integración con Tonomo

Conecta AutoHDR con Tonomo para optimizar tu flujo de trabajo de edición de fotos.

## Configuración

Contacta a [delivery@autohdr.com](mailto:delivery@autohdr.com) para instrucciones de
configuración de integración específicas para tu cuenta de Tonomo.

## Cómo funciona

Una vez configurado, Tonomo envía tus fotos a AutoHDR para procesamiento.
Las fotos editadas se entregan de vuelta a tu proyecto de Tonomo automáticamente.
