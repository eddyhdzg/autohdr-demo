# Integración con HD PhotoHub

Conecta AutoHDR con HD PhotoHub para optimizar tu flujo de trabajo de edición de fotos.

## Configuración

Contacta a [delivery@autohdr.com](mailto:delivery@autohdr.com) para instrucciones de
configuración de integración específicas para tu cuenta de HD PhotoHub.

## Cómo funciona

Una vez configurado, HD PhotoHub envía tus fotos a AutoHDR para procesamiento.
Las fotos editadas se entregan de vuelta a tu proyecto de HD PhotoHub automáticamente.
