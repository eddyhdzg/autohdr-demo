# AutoHDR Integration Skill

AutoHDR is an AI-powered real estate photo editing API. This skill helps you integrate AutoHDR into applications.

## Key Concepts
- Credit-based pricing: users only pay for downloaded photos
- Upload photos → AI edits → preview → download
- Supports: HDR editing, flambient, day-to-dusk, grass greening, object removal, virtual staging, TV blackout

## Integrations
- Spiro, HD PhotoHub, Tonomo, Aryeo, Dropbox
- See integration docs for setup guides

## Common Tasks
- Upload photos via the dashboard or connected integration
- Preview and compare before downloading
- Buy extra credits at your plan's per-photo rate
- Set up automated workflows through integrations
